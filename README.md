## Installation

### Prerequisites

  Node.js and npm installed (check with `node -v` and `npm -v`).

### Setup Instructions

1. Clone this repository:
    ```bash
    git clone https://github.com/Vimal567/appscrip-task-vimal-s.git
    cd Appscrip-task-vimal-s
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Start the app:
    ```bash
    npm start
    ```
